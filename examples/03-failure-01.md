# Pattern 3: ~하며/~하고 피상적 분석 — 실패 사례 (오탐)

## 입력 텍스트

> 회사는 원자재 수입 비용을 줄이며 국내 생산 비중을 높였다. 그 결과 2024년 3분기 영업이익률이 전년 대비 4.2%p 개선됐다.

## 기대 출력

> (교정 없음 — 패턴 3이 발화되어서는 안 되는 텍스트)

## 적용 패턴

- Pattern 3 (~하며/~하고 피상적 분석): "줄이며"라는 "~하며" 연결형이 사용되어 표면적으로 해당 패턴처럼 보일 수 있음.

## 판정

**실패 (오탐)** — "~하며" 연결형이 1개만 사용되어 발화 기준인 3개에 한참 못 미친다. 또한 제외 조건인 "실제 인과·시간 관계를 나타내는 단일 ~하며 연결"에 해당한다. "비용을 줄이며 → 생산 비중을 높였다 → 영업이익률 4.2%p 개선"이라는 명확한 인과 관계가 있고, 구체적 수치가 뒤따르고 있으므로 피상적 나열이 아니다. 이 텍스트에서 패턴 3을 발화시키면 오탐이다.
