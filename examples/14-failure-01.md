# Pattern 14: 볼드체 남발 — 실패 사례 (오탐)

## 입력 텍스트

> 설정 화면에서 **저장** 버튼을 클릭한 뒤, 상단의 **미리보기**를 눌러 결과를 확인하세요. 변경 사항을 취소하려면 **취소**를 누르고, 이전 버전으로 돌아가려면 **되돌리기**를 선택하면 됩니다.

## 기대 출력

> (교정 없음 — 패턴 14가 발화되어서는 안 되는 텍스트)

## 적용 패턴

- Pattern 14 (볼드체 남발): **저장**, **미리보기**, **취소**, **되돌리기** 등 볼드 4개가 한 문단에 있어 표면상 발화 조건(3개 이상)에 도달함.

## 판정

**실패 (오탐)** — 이 텍스트는 UI 조작 가이드 문서이며, 볼드 처리된 단어는 모두 실제 버튼명이다. 제외 조건("UI 문서에서 버튼명·필드명 표시")에 정확히 해당한다. 사용자가 화면에서 찾아야 할 인터페이스 요소를 시각적으로 구분하기 위한 볼드이므로, 이를 제거하면 오히려 가이드의 가독성이 떨어진다.
