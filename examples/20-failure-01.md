# Pattern 20: 학습 데이터 기한 면책 — 실패 사례 (오탐)

## 입력 텍스트

> GPT-4의 학습 데이터는 2023년 4월 기준이며, 이후 발생한 사건이나 변경 사항은 반영되지 않았다. 이러한 학습 데이터 기한 제약은 모든 대규모 언어 모델이 공유하는 구조적 한계다.

## 기대 출력

> (교정 없음 — 패턴 20이 발화되어서는 안 되는 텍스트)

## 적용 패턴

- Pattern 20 (학습 데이터 기한 면책): "학습 데이터는 2023년 4월 기준", "반영되지 않았다", "학습 데이터 기한 제약"이 등장.

## 판정

**실패 (오탐)** — 이 텍스트는 AI 시스템 자체를 다루는 기술 문서이며, GPT-4라는 특정 모델의 학습 데이터 한계를 객관적으로 설명하고 있다. Pattern 20이 탐지하는 것은 편집·보도·분석 글에서 AI가 자신의 한계를 독자에게 슬쩍 전가하는 면책 표현이지, AI 기술을 주제로 한 문서에서 모델 사양을 정확히 기술하는 것이 아니다. 여기서 "학습 데이터 기한"은 면책이 아니라 분석 대상이므로 교정 대상이 아니다.
