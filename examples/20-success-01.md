# Pattern 20: 학습 데이터 기한 면책 — 성공 사례

## 입력 텍스트

> 삼성전자는 반도체와 스마트폰 사업을 양대 축으로 성장해 온 글로벌 기업이다. 구체적인 정보는 제한적이나, 확인 가능한 자료에 따르면 매출 규모는 약 300조 원 수준으로 알려져 있다. 다만 이 수치는 2023년 기준으로, 최신 정보와 다를 수 있으므로 참고용으로만 활용하기 바란다.

## 기대 출력

> 삼성전자는 반도체와 스마트폰 사업을 양대 축으로 성장해 온 글로벌 기업이다. 2023년 사업보고서 기준 연결 매출은 약 258.9조 원이었다.

## 적용 패턴

- Pattern 20 (학습 데이터 기한 면책): "구체적인 정보는 제한적이나", "확인 가능한 자료에 따르면", "최신 정보와 다를 수 있으므로" — 기업 분석 글에서 AI 학습 데이터 한계를 암시하는 면책 표현이 3개 등장한다.

## 판정

**성공** — 편집·분석 콘텐츠에서 AI가 자신의 학습 데이터 한계를 독자에게 전가하는 전형적인 경우다. "구체적인 정보는 제한적이나"는 AI가 학습 데이터에 해당 기업 정보가 부족할 때 쓰는 자기 참조 표현이고, "최신 정보와 다를 수 있으므로"는 학습 데이터 기한 면책의 대표적 문구다. 교정 후 면책 문구를 제거하고 출처와 연도를 명시한 구체적 수치로 대체했다.
