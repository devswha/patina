# Pattern 21: 아첨하는 말투 — 실패 사례 (과교정)

## 입력 텍스트

> 인터뷰이: "좋은 질문이네요. 사실 그 부분은 저도 많이 고민했습니다. 창업 초기에는 시장 자체가 존재하지 않았거든요."

## 기대 출력

> (교정 없음 — 패턴 21이 발화되어서는 안 되는 텍스트)

## 적용 패턴

- Pattern 21 (아첨하는 말투): "좋은 질문이네요"가 등장.

## 판정

**실패 (과교정)** — "좋은 질문이네요"가 표면적으로 아첨 표현처럼 보이지만, 이는 인터뷰 전사본에서 실제 인터뷰이가 발화한 직접 인용이다. Pattern 21이 제거하려는 것은 AI가 사용자 비위를 맞추기 위해 본론 앞에 삽입하는 기계적 칭찬이지, 실제 사람이 대화 중에 쓴 자연스러운 반응이 아니다. 인터뷰 전사본의 직접 인용을 교정하면 원문의 사실성이 훼손된다.
