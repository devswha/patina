# Pattern 22: 채움 표현 — 실패 사례 (오탐)

## 입력 텍스트

> 납품일로부터 10일 이내에 대금을 지급하는 경우에 한하여 3% 할인이 적용된다. 단, 이 할인은 계약 금액이 1억 원을 초과하는 경우에 한하여 유효하다.

## 기대 출력

> (교정 없음 — 패턴 22가 발화되어서는 안 되는 텍스트)

## 적용 패턴

- Pattern 22 (채움 표현): "~하는 경우에 한하여"가 2회 등장.

## 판정

**실패 (오탐)** — "경우에 한하여"가 표면적으로는 "~면"으로 축약 가능한 채움 표현처럼 보이지만, 이는 계약서의 법률 문맥이다. "납품일로부터 10일 이내에 대금을 지급하면 할인"이라고 바꾸면 법적 구속력이 모호해진다. Pattern 22의 제외 조건에 명시된 대로, 법률·규정 문서에서 정밀성을 위해 필요한 표현은 채움 표현으로 간주하지 않는다. "경우에 한하여"는 적용 범위를 엄격하게 제한하는 법률 용어이며, 삭제 시 의미가 달라진다.
