# Pattern 28: 불필요한 외래어 남발 — 성공 사례 02

## 입력 텍스트

> 블랙박스 패러프레이저도, 탐지기 우회 약속도 아닙니다. patina는 패턴 기반이고 감사 가능해서, 무엇을 왜 바꿨는지와 원문의 주장이 보존됐는지를 보여줍니다.

## 기대 출력

> 블랙박스형 재작성 도구도, AI 탐지기 우회 도구도 아닙니다. patina는 패턴을 기준으로 고친 부분과 이유, 원문 주장이 유지됐는지를 보여줍니다.

## 적용 패턴

- Pattern 28 (불필요한 외래어 남발): 공개 README의 핵심 설명에서 "패러프레이저"가 독자층을 좁히는 외래어로 쓰였다.
- Pattern 26 (번역체): "탐지기 우회 약속"은 detector-bypass promise를 직역한 압축 명사구라 한국어 문장 안에서 어색하다.

## 판정

**성공** — 외래어를 기계적으로 모두 순화하지 않고, 독자가 바로 이해할 수 있는 "재작성 도구"와 "AI 탐지기 우회 도구"로 풀었다. "우회 도구가 아니다"라는 윤리적 부정 의미도 유지했다.
