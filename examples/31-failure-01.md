# Pattern 31: 결론 신호어 남용 — 실패 사례 (오탐)

## 입력 텍스트

> ## 4. 결론
>
> 본 연구에서는 한국형 LLM의 한국어 추론 성능 향상을 위한 새로운 fine-tuning 방법을 제안하였다. 실험 결과, 제안된 방법은 기존 baseline 대비 BLEU 4.2점, ROUGE-L 3.7점 향상을 달성하였다.
>
> 결론적으로, 본 연구의 contribution은 (1) 한국어 instruction tuning에 특화된 데이터셋 구축 방법, (2) 적은 파라미터로도 효과적인 PEFT 변형, (3) 검증된 평가 protocol 제안이다.

## 기대 출력

> (교정 없음 — 패턴 31이 발화되어서는 안 되는 텍스트)

## 적용 패턴

- Pattern 31 (결론 신호어 남용): 마지막 문단이 "결론적으로"로 시작.

## 판정

**실패 (오탐)** — 이 텍스트는 학술 논문의 "## 4. 결론" 섹션에 해당하며, Pattern 31의 제외 조건 "학술 논문의 '결론' 섹션 헤더 직후 첫 문단"에 해당한다. 헤더가 이미 결론을 명시적으로 표시했지만, 본문에서 한 번 더 "결론적으로"로 contribution 목록을 도입하는 것은 학술 논문의 정형화된 관례다. 이를 평서문으로 바꾸면 contribution 목록이라는 신호가 약해지고 가독성이 떨어진다.
