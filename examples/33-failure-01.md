# Pattern 33: 정의형 은유 등식 — 실패 사례 (오탐)

## 입력 텍스트

> 이 API는 인증 시스템의 심장이다. 모든 요청이 여기서 토큰을 검증하고, 지난달 호출량은 3억 건이었다.

## 기대 출력

> (교정 없음 — 패턴 33이 발화되어서는 안 되는 텍스트)

## 적용 패턴

- Pattern 33 (정의형 은유 등식): `API는 ... 심장이다`라는 단일 은유 등식 1회.

## 판정

**실패 (오탐)** — 단일 1회 사용이고, 바로 뒤에 "모든 요청이 여기서 토큰을 검증"하고 "지난달 호출량은 3억 건"이라는 구체 근거가 붙어 있어 제외 조건에 해당한다. 이 문장은 막연한 금언이 아니라 시스템 병목과 중요도를 설명하는 기술적 비유다.
