# Pattern 34: 무생물 행위주 — 실패 사례 (오탐)

## 입력 텍스트

> 본 연구는 대규모 언어 모델의 한국어 종결어미 분포를 분석한 결과를 보여준다. 실험 데이터는 세 가지 코퍼스에서 수집되었다. 분석 결과는 모델 계열 간 유의한 차이를 시사한다. 기온이 상승하면서 실험실 장비의 오차율도 함께 올라갔다.

## 기대 출력

> (교정 없음 — 패턴 34가 발화되어서는 안 되는 텍스트)

## 판정 근거

- "본 연구는 ~을 보여준다", "분석 결과는 ~을 시사한다" — 학술 인용 관행(저자에 대한 관용적 환유), 제외 조건 해당.
- "기온이 상승하면서" — 실제로 행위자 없는 자연 과정.
- 무생물 주어 문장이 여러 개지만 전부 제외 조건에 속하므로 발화 금지. 이를 교정하면 학술 문체 파괴 + 과편집.
