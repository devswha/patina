# Pattern 35: 前置き咳払い — Success

## Input Text

> 正直に言うと、解約の根本原因は価格ではありません。はっきり言って、ユーザーは最初の一週間で、請求書を見る前に離脱します。ぶっちゃけ、解約を生んでいるのは課金ではなくオンボーディングです。

## Expected Output

> 解約の根本原因は価格ではない。ユーザーは最初の一週間で、請求書を見る前に離脱する。解約を生んでいるのは課金ではなくオンボーディングだ。

## Applied Pattern

- Pattern 35 (前置き咳払い): 一段落に前置き咳払いが 3 つ——「正直に言うと」「はっきり言って」「ぶっちゃけ」。いずれも次の文を率直な発言として演出するだけで、内容を何も足していない。

## Judgment

**Success** — 発火条件を満たす：前置き 3 回は 2 回以上の閾値を大きく超え、どの場合も前置きの直後が本論そのものである。削っても失われる内容はなく、主張はより速く、より率直に届く。
