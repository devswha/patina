# Pattern 37: コロン種明かし — Success

## Input Text

> 連休中にインポーターを Rust で書き直した。結果：12 倍の高速化。一番すごいのは：メモリ使用量が半減。オチ：バイナリは元の Python バンドルより小さい。

## Expected Output

> 連休中にインポーターを Rust で書き直して 12 倍速くなった。メモリ使用量は半減し、バイナリは元の Python バンドルより小さい。

## Applied Pattern

- Pattern 37 (コロン種明かし): 4 文中にコロン種明かしが 3 つ——「結果：」「一番すごいのは：」「オチ：」——事実の一つ一つをオチの供給に変えている。

## Judgment

**Success** — 発火条件を満たす：コロン演出 3 回は 2 回以上の閾値を超え、どのコロンもリスト・ラベル・定義の構造的役割を持たない。平叙文が同じ事実をドラムロールなしで伝え、段落はエンゲージメント最適化コピーのようには読めなくなる。
